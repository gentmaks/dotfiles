# Dotfiles for Neovim, Ghostty, etc (updated continuously)

Personal configuration files for my development environment.

## Structure

- `nvim/` – Neovim config (`init.lua`, plugins, keymaps, and options)
- `gh/` – GitHub CLI configuration
- `ghostty/` – Ghostty terminal config and themes
- `raycast/` – Raycast extensions and AI scripts

## Installation

Clone the repo:

```sh
git clone https://github.com/<username>/dotfiles ~/.dotfiles
```
